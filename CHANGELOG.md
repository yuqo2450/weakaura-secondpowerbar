# Changelog

All notable changes to this project will be documented in this file.

## [v10.2.0] - 2023-11-07

### Changed

* Updated import string for game version 10.2.0
* Using game versions for aura versions

### Added

### Added

* Stagger as a resource for brewmaster monks

### Changed
